Selected project

Doorbell delivery platform

Repository

Doorbell is a multi-application food delivery platform with separate experiences for customers, riders, merchants, and administrators. The project brings together backend services, shared API contracts, mobile applications, and web dashboards.

What it demonstrates

  • End-to-end product thinking across multiple user roles.
  • Backend and frontend integration through typed API models.
  • Mobile workflows for customer ordering and rider delivery.
  • Administrative interfaces for operational control.

Technologies

Java, Spring Boot, TypeScript, React Native, Angular, REST APIs, Firebase, OpenAPI, Node.js.
